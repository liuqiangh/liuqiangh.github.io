OS库的基本使用

os库提供了通用的、基本的操作系统交互功能，包括路径操作、进程管理、环境参数等几大功能。

路径操作

os.path子库以path为入口，用于操作和处理文件路径。

os.path.abspath(path):返回path在当前系统中的绝对路径；
os.path.relpath(path):返回path与当前程序之间的相对路径；
os.path.exists(path):判断path对应文件或目录是否存在，返回True或False;
